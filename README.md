# 🚀 WorkPulse – Smart Productivity Tracker

A SaaS-style task tracking and productivity analysis tool built using Python and Streamlit.

---

## 🔥 Features

### 👨‍💻 Employee Module
- Start / Pause / Stop task tracking
- Predefined tasks:
  - List Bill Audit
  - Retro Bill Audit
  - CPS Creation
- Automatic time logging
- Tracks Active vs Idle time

### 👨‍💼 Manager Dashboard
- Filter by:
  - Employee
  - Task
  - Date
- View productivity metrics
- Download reports (CSV)

---

## 📊 Key Metrics
- Total Time
- Idle Time
- Active Work Time

---

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
